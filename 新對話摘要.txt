## 技術架構

### 前端（GitHub Pages）
純靜態 HTML + JS，無框架。**重要：不使用 supabase-js client**，改用直接 fetch 呼叫 Supabase REST API（因 supabase-js 在此環境下 apikey header 無法正常帶入）。

**Supabase fetch 標準寫法（所有頁面統一）：**
```javascript
const SUPABASE_URL = 'https://kldumnxdlgovaxtiszyn.supabase.co';
const SUPABASE_ANON = 'eyJhbGci...（完整 key）';
const SB_HEADERS = {
  'apikey': SUPABASE_ANON,
  'Authorization': 'Bearer ' + SUPABASE_ANON,
};

async function sbQuery(table, params = {}) {
  let url = SUPABASE_URL + '/rest/v1/' + table + '?';
  const qs = [];
  if (params.select) qs.push('select=' + encodeURIComponent(params.select));
  if (params.eq) for (const [col, val] of Object.entries(params.eq)) qs.push(col + '=eq.' + encodeURIComponent(val));
  if (params.neq) for (const [col, val] of Object.entries(params.neq)) qs.push(col + '=neq.' + encodeURIComponent(val));
  if (params.gte) for (const [col, val] of Object.entries(params.gte)) qs.push(col + '=gte.' + encodeURIComponent(val));
  if (params.order) for (const [col, asc] of params.order) qs.push('order=' + col + '.' + (asc ? 'asc' : 'desc'));
  if (params.limit) qs.push('limit=' + params.limit);
  if (params.offset) qs.push('offset=' + params.offset);
  url += qs.join('&');
  try {
    const resp = await fetch(url, { headers: SB_HEADERS });
    if (!resp.ok) return { data: [], error: resp.statusText };
    const data = await resp.json();
    return { data: Array.isArray(data) ? data : [], error: null };
  } catch(e) {
    return { data: [], error: e.message };
  }
}
```

## 台股主力雷達（radar.html）

### 三個頁籤
1. **🆕 主力新建倉** — 本月相較上月新增個股（投信）+ 外資連續買超
2. **📊 主力持股追蹤** — 三個子頁：多方共識 / 依基金查詢 / 法人清單
3. **🧩 主力ETF追蹤** — 投信持有 ETF 清單，依月份和類型篩選

### 資料來源
- 投信持股：SITCA 每月前十大持股（月更）
- ETF 判斷：`/^0\d{3,5}[A-Za-z]?$/`（0開頭4-6碼）
- 新建倉定義：本月相較上月新增（`n=1`）

### taiwan_scanner.py 重要邏輯
- ETF 過濾：`is_etf()` 函式，新建倉和外資訊號都排除 ETF
- 歷史補抓：`--month YYYYMM` 參數，強制重爬（`if check.data and not target_month`）
- 比對月份：`get_prev_months(current_month, n=1)`，只比對上個月

## 新聞系統（news.html）

### 架構
- **兩個頁籤**：🇹🇼 台股日報 / 🇺🇸 美股日報
- **資料來源**：NewsAPI + GNews API 自動抓取，每天早上 7:30 更新
- **儲存**：`daily_news` 表，`lang` 欄位標記 zh/en
- **前端**：英文新聞顯示 `🇺🇸 英文` 標籤

### news_scanner.py 邏輯
- **台股**：GNews 台灣在地中文優先 → NewsAPI 中文補充 → 英文補充
- **美股**：GNews 中文優先 → NewsAPI 中文補充 → 英文補充
- **過濾**：`is_finance_news()` 排除娛樂/非財經新聞
- **語言判斷**：含中文字元 → `lang=zh`，否則 `lang=en`

### GitHub Secrets 需要
- `SUPABASE_URL`、`SUPABASE_KEY`
- `NEWS_API_KEY`（newsapi.org，免費100次/天）
- `GNEWS_API_KEY`（gnews.io，免費100次/天）

## 開發注意事項

### 關鍵限制
- **不要用 supabase-js CDN**：在此環境 apikey header 無法正常帶入，一律改用直接 fetch
- **不要用 `js/supabase.js` 的 `sb` 物件**：同上原因，新頁面不要依賴它
- JS 字串內禁止用全形字元（會造成 SyntaxError）
- GitHub Actions 版本要用 v4（v3 已棄用）
- Yahoo Finance 需透過 CORS proxy 抓取

### Supabase RLS 設定
- `tw_fund_positions`：anon read（SELECT），已設定 policy
- `daily_news`：anon read + anon insert + anon delete，已設定 policy
- 新建表後記得設定 RLS policy，否則 anon key 無法讀寫
